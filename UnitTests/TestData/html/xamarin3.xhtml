Today we are very excited to announce Xamarin 3, which includes four major advances in mobile development:
Xamarin is proud to unveil the world's best user interface designer for iOS, integrated into both Xamarin Studio and Visual Studio. Build beautiful native user interfaces, including live preview of custom controls, from the comfort of your favorite IDE. Learn more »
Xamarin.Forms is a simple API that allows developers to quickly and easily build native iOS, Android and Windows Phone user interfaces from a single, shared C# codebase. Delivered as a portable class library, Xamarin.Forms includes more than 40 controls and layouts and makes it easy to mix and match your shared UI code with the native, platform-specific user interface capabilities Xamarin has always given you. Learn more »
Xamarin 3 introduces Shared Projects and full support for Portable Class Libraries — two great new code-sharing techniques for cross-platform apps. Learn more »
Xamarin now includes a major user experience refresh, support for NuGet packages, full .NET BCL documentation, source analysis, F# support, streamlined Visual Studio support, and many other great features. Learn more »
These are just four of the hundreds of new features and enhancements in Xamarin 3.
