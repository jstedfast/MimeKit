Remarks

Gets or sets the root document that references the other MIME parts within the multipart/related.

When getting the root document, the "start" parameter of the Content-Type header is used to determine which of the parts is the root. If the "start" parameter does not exist or does not reference any of the child parts, then the first child is assumed to be the root.

When setting the root document MIME part, the Content-Type header of the multipart/related part is also updated with a appropriate "start" and "type" parameters.
