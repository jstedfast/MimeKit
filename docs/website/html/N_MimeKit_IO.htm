MimeKit.IO Namespace
The MimeKit.IO namespace provides classes that extend the System.IO namespace.
Classes
  ClassDescription
BoundStream
A bounded stream, confined to reading and writing data to a limited subset of the overall source stream.

MeasuringStream
A stream useful for measuring the amount of data written.
MemoryBlockStream
An efficient memory stream implementation that sacrifices the ability to get access to the internal byte buffer in order to drastically improve performance.
