Remarks

Creates a new SqliteCertificateDatabase and opens a connection to the SQLite database at the specified path using the Mono.Data.Sqlite binding to the native SQLite library.

If Mono.Data.Sqlite is not available or if an alternative binding to the native SQLite library is preferred, then consider using SqliteCertificateDatabase(IDbConnection, String) instead.
